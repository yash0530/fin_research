# Backtest Reports

This directory contains JSON files with reports generated by the backtest job (`npm run job -- backtest`).
Each report is saved with a timestamped filename: `backtest-<YYYY-MM-DD-HHmmss>.json`.

## Structure of the JSON output:
- `timestamp`: The date/time when the backtest was run.
- `startISO`: The start date of the backtest grid (default 2010-01-01).
- `endISO`: The end date of the backtest grid (default maxPriceDate - 400 days).
- `horizons`: The forward return horizons evaluated (21d, 63d, 126d, 252d).
- `summary`: The aggregate results for each family and horizon across all periods (N, flagged mean, baseline mean, excess, hit rate).
- `runs`: Month-by-month results for each family and horizon.
