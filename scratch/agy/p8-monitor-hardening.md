## Build

Output top-3 changed paragraph pairs `{section, before, after, jaccard}` + counts. Pure over injected doc texts; fixture-tested with synthetic 10-K text pairs (boilerplate filtered, real change surfaces, <3 changes OK).

Persist per-symbol results as `FilingEvent` rows kind `filing-diff` (severity from LLM verdict: routine/notable/thesis-relevant) + write into the run artifact. Steps checkpoint/resume like other run types. Remove the "not yet implemented" placeholder.

### Built

1. **`src/monitor/filing-diff.ts`** (+ test, 10 cases): pure core — `stripHtml` → `splitParagraphs` (section-heading tracking) → regex boilerplate blocklist (safe-harbor / forward-looking / PSLRA / ASC-ASU recitals) → per-paragraph best-match alignment (same section first, whole doc fallback) → Jaccard on stopword-filtered token sets. Changed = Jaccard < 0.6 AND company-token gate (ticker / digits / mid-sentence capitalized multi-word product nouns). Near-verbatim (≥0.9) = unchanged. Returns top-3 `{section, before, after, jaccard}` + honest counts. Fixture-tested on synthetic 10-K pairs: boilerplate stripped, real risk-factor/MD&A change surfaces, generic-prose churn gated out, top-3 cap with `changedCount=5`, <3 changes OK.
2. **`filing_diff` run type** (`src/runs/runner.ts`, placeholder removed): plans one resumable step per watchlist+portfolio symbol (`maxTickers` budget-scaled). Each step: pick two most recent 10-K (else 10-Q) filings from `EdgarFiling`, fetch primary docs via the shared `EDGAR_LIMITER` (200KB cap; injectable `fetchImpl`/`userAgent`), run the pure core, then `completeJson(providerFor("narrator"))` — fast profile — over ONLY the changed paragraphs (accessionNo provenance in prompt + report). Persists `FilingEvent` kind `filing-diff`, item `diff`, severity = LLM verdict (`routine|notable|thesis-relevant`); upsert on `(accessionNo,item)`. Never-crash: per-symbol failures recorded in the checkpoint; invalid LLM output degrades to a deterministic summary (severity `notable`); zero changes skips the LLM entirely. Artifact writes a batch table + per-symbol quote blocks. Runner tests (fake fetch + FakeProvider): FilingEvent persisted with verdict severity, boilerplate never reaches the model, skip path (<2 filings), bad-LLM degradation, empty-watchlist no-targets step.
